## Requirements

- Python 3.10 or higher
- pip or pipx

## Installation

<Tabs>
  <Tab title="pip">
    ```bash
    pip install cpm
    ```
  </Tab>
  <Tab title="pipx (Recommended)">
    [pipx](https://pypa.github.io/pipx/) installs CLI tools in isolated environments:

    ```bash
    pipx install cpm
    ```
  </Tab>
  <Tab title="From Source">
    ```bash
    git clone https://github.com/openlit/cpm.git
    cd cpm/python
    pip install -e .
    ```

    For development with test dependencies:

    ```bash
    pip install -e ".[dev]"
    ```
  </Tab>
</Tabs>

## Verify Installation

```bash
cpm --version
```

## Dependencies

The Python SDK uses the following dependencies:

| Package | Purpose |
|---------|---------|
| [click](https://click.palletsprojects.com/) | CLI framework |

## Troubleshooting

### Command not found

If `cpm` is not found after installation, ensure pip's bin directory is in your PATH:

```bash
# Check where pip installs scripts
python -m site --user-base

# Add to PATH (bash/zsh)
export PATH="$PATH:$(python -m site --user-base)/bin"
```

### Permission errors

Use `--user` flag or pipx to avoid permission issues:

```bash
pip install --user cpm
# or
pipx install cpm
```

### Python version

CPM requires Python 3.10+. Check your version:

```bash
python --version
```
